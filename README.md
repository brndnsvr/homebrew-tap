# Homebrew Tap

Personal Homebrew tap for macOS applications.

## Installation

```bash
brew tap brndnsvr/tap
```

## Available Casks

### WakeyWakey

Menu bar app that keeps your Mac awake with subtle mouse movements.

```bash
brew install --cask wakeywakey
```

[More info](https://github.com/brndnsvr/WakeyWakey)
