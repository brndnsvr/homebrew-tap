# Homebrew Tap

Personal Homebrew tap for macOS tools and applications.

## Installation

```bash
brew tap brndnsvr/tap
```

## Available Formulae

### sysm

Unified CLI for Apple ecosystem integration on macOS.

```bash
brew install sysm
```

[More info](https://github.com/brndnsvr/sysm)

## Available Casks

### WakeyWakey

Menu bar app that keeps your Mac awake with subtle mouse movements.

```bash
brew install --cask wakeywakey
```

[More info](https://github.com/brndnsvr/WakeyWakey)
